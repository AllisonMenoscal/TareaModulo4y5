¿Que es la navegación?

Tambien conocido como enrutamiento es un mecanismo en las aplicaciones web que le permite a un usuario buscar contenido en la vista según el componente que desee navegar. También se puede hacer directamente utilizando la URL de ruta en los navegadores.


Angular para procesar las rutas y determinar cual será la vista a presentar utiliza un modulo de enrutado raíz

AppRoutingModule

Cuando se crea un proyecto en angular se crea por ddefault el modulo raiz que permite manejar el enrutamiento. este modulo hace uso del RouterModule, esta contiene un array de routes e instrucciones que se ejecutan cuando dichas rutas se activan

Router Link

En concreto esta directiva, que tambien viene en el modulo routerModule, se usa en sustitución del atributo estándar href


Router Link Active

Para configurar los enrutadores en clase Angular y activa, se lo realiza con las directivas routerLink y routerLinkActive
